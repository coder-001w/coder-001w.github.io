Entity Framework Core基础

entity framework是微软官方的ORM框架，所谓ORM（object relational mapping，对象关系映射）中的对象即使C#中的对象，关系指的是关系数据库，如mysql，sqlserver等，映射是指在关系数据库与C#对象之间架起一座桥梁，让我们只需要操作C#对象就可以将数据插入数据库，而不需要sql语句。
